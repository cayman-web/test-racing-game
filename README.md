# GT3 Racer — Porsche 992 GT3 R Test

Мобильная 2D-гонка вид сверху. Управление — тач-кнопки в углах экрана.

## Как выложить на GitHub Pages и открыть с айфона

1. Создай новый репозиторий на GitHub (например `gt3-racer`).
2. Залей туда всё содержимое этой папки как есть: `index.html`, `style.css`,
   `track.js`, `car.js`, `render.js`, `main.js`, `manifest.json`, папку `icons/`.
   Если хочешь свою текстуру машины — положи туда же файл `car.png`
   (нос машины на фото должен смотреть вверх/на север).
3. В репозитории: **Settings → Pages → Source** выбери ветку `main` и папку `/ (root)`, сохрани.
4. Подожди 1-2 минуты, GitHub покажет ссылку вида
   `https://<твой-логин>.github.io/gt3-racer/`.
5. Открой эту ссылку в Safari на iPhone.
6. Нажми кнопку **"Поделиться"** (квадрат со стрелкой вверх) → **"На экран «Домой»"**.
7. Запускай игру с иконки на главном экране — откроется в полноэкранном
   режиме, без адресной строки Safari, как обычное приложение.

Игра рассчитана на альбомную (горизонтальную) ориентацию телефона — если
держать вертикально, появится подсказка повернуть экран.

## Управление

- Низ-лево — руль (◀ ▶)
- Низ-право — газ / тормоз
- Верх-право — зум (+ / −)
- На компьютере то же самое работает с клавиатуры: WASD/стрелки, R — сброс,
  колесо мыши — зум.
