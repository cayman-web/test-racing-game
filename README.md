# GT3 Racer — Porsche 992 GT3 R Test

Мобильная 2D-гонка вид сверху с главным меню, выбором трассы, статистикой
пилота и настройками.

## Структура

- `index.html` + `menu.js` + `menu.css` — главное меню (Drive / Your Statistics / Settings)
- `track-select.html` + `track-select.js` — выбор трассы (открывается через «сменить трассу» на кнопке Drive)
- `race.html` — сама гонка (открывается по кнопке Drive; можно передать `?track=id`)
- `stats.html` + `stats.js` — статистика пилота
- `settings.html` + `settings.js` — настройки (приоритетный метод управления)
- `style.css`, `track.js`, `car.js`, `render.js`, `main.js` — движок и HUD гонки
- `manifest.json`, `icons/` — для установки как приложение на iPhone

## Трассы

Все трассы описаны в реестре `TRACKS` в начале `track.js`. Сейчас их две:

- **Sketch Circuit** — первая трасса, классический лэйаут без гравия и питлейна.
- **Esses & Hook** — вторая трасса по наброску с эсками и шпилькой:
  - **Гравийно-песочные зоны вылета** — попадание в них гораздо суровее травы
    (сильно гасят скорость и почти не дают сцепления), и всегда считаются
    нарушением лимитов трассы (круг не засчитается как чистый).
  - **Пит-лейн** — рабочий (не декоративный): по нему можно физически проехать,
    он не считается сходом с трассы, но там действует лимит скорости (80 км/ч
    по умолчанию) — превысил лимит - круг тоже не засчитается, в HUD появится
    предупреждение.

Кнопка **Drive** сразу запускает последнюю выбранную трассу; маленькая кнопка
**«сменить трассу»** в углу той же плитки открывает экран выбора со всеми
доступными трассами и превью их layout.

Добавить свою трассу — значит дописать новый объект в `TRACKS` в `track.js`
(опорные точки контура в метрах, при желании `gravelZones` и `pitlane`) -
остальное (меню, превью, физика, HUD) подхватит её автоматически.

## Свои текстуры (опционально, кладутся рядом с index.html)

- `car.png` — текстура машины (нос должен смотреть вверх/на север). Используется
  и в самой гонке, и как превью на кнопке Drive в меню.
- `track-bg.jpg` — своя фотография/текстура трассы для фона кнопки Drive.
  Если файла нет — рисуется приглушённый контур трассы автоматически, ничего
  дополнительно готовить не нужно.

## Как выложить на GitHub Pages и открыть с айфона

1. Создай новый репозиторий на GitHub (например `gt3-racer`).
2. Залей туда всё содержимое этой папки как есть.
   Если хочешь свою текстуру машины/трассы — положи `car.png` и/или
   `track-bg.jpg` туда же.
3. В репозитории: **Settings → Pages → Source** выбери ветку `main` и папку `/ (root)`, сохрани.
4. Подожди 1-2 минуты, GitHub покажет ссылку вида
   `https://<твой-логин>.github.io/gt3-racer/`.
5. Открой эту ссылку в Safari на iPhone.
6. Нажми кнопку **"Поделиться"** (квадрат со стрелкой вверх) → **"На экран «Домой»"**.
7. Запускай игру с иконки на главном экране — откроется в полноэкранном
   режиме, без адресной строки Safari, как обычное приложение. Попадёшь сразу
   в меню.

Сама гонка (`race.html`) рассчитана на альбомную (горизонтальную) ориентацию —
если держать телефон вертикально, появится подсказка повернуть экран. Меню
работает в любой ориентации.

## Управление в гонке

- Низ-лево — руль (◀ ▶)
- Низ-право — газ / тормоз
- Верх-право — зум (+ / −)
- Bluetooth-геймпад: левый стик — руль, RT/R2 — газ, LT/L2 — тормоз. При
  активности геймпада кнопки на экране плавно прячутся, любое касание экрана
  их возвращает.
- На компьютере то же самое работает с клавиатуры: WASD/стрелки, R — сброс,
  колесо мыши — зум.
- В настройках можно выбрать, что показывать при запуске заезда по умолчанию —
  экранные кнопки или сразу режим геймпада.

## Память и статистика

Игра запоминает (через localStorage в браузере, только на этом устройстве):
последнюю выбранную трассу и машину, а также общий пробег, число кругов и
соотношение чистых/грязных (по нарушению лимитов трассы или лимита скорости
на питлейне) — всё это видно на экране Your Statistics.
